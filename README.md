# opac_search
Pythonの軽量WebフレームワークであるFlaskを用いてWebアプリケーションを設計・実装し、SQLiteデータベースと連携した全文検索システムを構築した。フロントエンドではHTMLフォームによるキーワード入力と検索対象フィールドの切り替え機能を実装し、バックエンドではPOSTリクエストの受信・パース、動的なSQL文の組み立て、クエリパラメータのバインディングによるSQLインジェクション対策を行った。また、markupsafeライブラリによる出力値のエスケープ処理を施し、XSS対策を含む基本的なWebセキュリティを意識した実装とした。実行環境にはGoogle Colaboratoryを採用し、クラウドストレージ（Google Drive）上のデータベースファイルをマウントして利用する構成とした。

なお、SQLiteデータベース（opaclarge.db）、検索フォームのHTMLテンプレート（report2.html）、Flask起動用モジュール（cje1gw）、および環境セットアップスクリプト（cje1install）については、著作権等の都合により本コードからは省略している。
